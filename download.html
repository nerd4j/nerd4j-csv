NERD4J-CSV

So far this is the quickest, easiest and most configurable open source Java framework for CSV manipulation.

Download

You can download the nerd4j-csv library and source code from our repository on GutHub.

This framework is also available on Maven Central here and can be used with the following dependecy declaration:

<dependency>
  <groupId>org.nerd4j</groupId>
  <artifactId>nerd4j-csv</artifactId>
  <version>1.3.0</version>
</dependency>

Nerd4j-csv is a very lightweight framework, it depends only on other 2 nerd4j libraries: nerd4j-core and nerd4j-utils, and from slf4j-api for logging purpose.
